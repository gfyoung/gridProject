About Off the Grid SF Search

The purpose of this web application is to facilitate the searching of food trucks and vendors in San Francisco. Inspired by an interview test for an internship, this application uses data from Off the Grid SF via the Facebook API to allow users to look up various trucks and vendors, monitor their recent activity, and find events that they will be catering in the near future. To learn more about Off the Grid SF, visit our Facebook page here or our Twitter page here. Otherwise, we hope you enjoy using the application!
